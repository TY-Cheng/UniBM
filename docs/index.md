# UniBM

UniBM packages the reusable statistical core for severity/extreme-value-index
and persistence/extremal-index inference under serial dependence.

This site focuses on the installable `unibm` package under `src/unibm`.
Repository orchestration for benchmarks, applications, and notebooks remains in
the root [`README.md`](https://github.com/TY-Cheng/UniBM/blob/main/README.md)
and [`justfile`](https://github.com/TY-Cheng/UniBM/blob/main/justfile).

## What this site covers

- package installation and local development setup
- the conceptual split between the severity and persistence branches
- small runnable examples for the public package surface
- the exported public namespaces `unibm`, `unibm.evi`, `unibm.ei`, and
  `unibm.cdf`
