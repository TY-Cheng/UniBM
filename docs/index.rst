UniBM Documentation
===================

UniBM packages the reusable statistical core for severity/EVI and
persistence/extremal-index inference under serial dependence. The benchmark,
application, and notebook workflows live under ``scripts/benchmark``,
``scripts/application``, ``scripts/shared``, and ``scripts/notebook_api.py``;
this site focuses on the installable ``unibm`` package under ``src/unibm``.
